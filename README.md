mouflon-scoop-bucket
=====================

Personal bucket for the scoop. (個人用Scoopバケット)

Setup
------

```cmd
scoop bucket add mouflon-scoop-bucket https://github.com/mouflon-jp/mouflon-scoop-bucket.git
```

Update buckets
---------------

```cmd
scoop update
```


For developers
----------------

### Update manifests

```ps
.\bin\checkver.ps1 -u
```
